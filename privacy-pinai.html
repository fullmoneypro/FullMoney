Komitmen Privasi & Keamanan Aplikasi Pinai

  • 100% Bebas Data Pribadi: Kami menghargai privasi Anda. Aplikasi ini tidak meminta email, nomor HP, atau nama asli Anda.
  • Percakapan Privat: Chat AI tidak disimpan di server kami. Setiap kali aplikasi dibuka, riwayat percakapan otomatis bersih.
  • Penyimpanan Lokal: Data fitur Catatan (Note) murni hanya disimpan di dalam memori perangkat (HP) Anda sendiri.
  • Sistem Kuota Awan & Akun Anonim: Saldo kuota hanya untuk akses fitur aplikasi dan Kuota Anda disimpan dengan aman di server cloud kami menggunakan ID Perangkat acak.
  • Fitur Brankas Kuota: Agar kuota Anda tidak hangus saat ganti HP atau Hapus Data, Anda dapat membuat Password di menu 'Akun'. Dengan ID Pinai dan Password, kuota Anda bisa dipulihkan kapan saja.
  • Penghapusan Akun & Data (Data Deletion): Anda memiliki hak penuh untuk meminta penghapusan akun dan data Anda (termasuk saldo kuota) dari server kami. Karena akun Anda bersifat anonim, Anda dapat mengajukan permohonan penghapusan dengan mengirimkan email ke fullmoneypro@gmail.com beserta ID Pinai Anda. Kami akan memproses penghapusan data secara permanen maksimal 1x24 jam.
  • Layanan Iklan (AdMob): Kami menggunakan Google AdMob untuk menyediakan fitur kuota gratis. Sistem ini menggunakan ID Iklan standar perangkat Anda secara anonim.
  • Pemutaran Video Pihak Ketiga: Aplikasi ini menggunakan pemutar video resmi dari Dailymotion (Embed) untuk menyajikan konten hiburan tanpa mengharuskan Anda untuk mendaftar (login). Dailymotion mungkin mengumpulkan data analitik anonim (seperti alamat IP) untuk penayangan iklan yang tunduk pada Kebijakan Privasi Dailymotion.
