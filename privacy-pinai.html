Komitmen Privasi & Keamanan Aplikasi Pinai

  • Bebas & Anonim: Aplikasi ini tidak mewajibkan login. Kami tidak mengumpulkan data pribadi seperti email atau profil Anda.
  • Percakapan Privat: Chat tidak disimpan di server kami. Setiap membuka aplikasi, riwayat chat otomatis kosong dan bersih.
  • Penyimpanan Lokal: Data fitur Catatan (Note) hanya disimpan secara lokal di dalam memori perangkat Anda.
  • Sistem Kuota Awan: Saldo kuota Anda disimpan dengan aman di server (cloud) menggunakan ID Perangkat anonim, bukan identitas pribadi.
  • Layanan Iklan (AdMob): Kami menggunakan Google AdMob untuk menyediakan fitur kuota gratis. Sistem ini mungkin menggunakan ID Iklan perangkat Anda secara anonim untuk menampilkan video yang sesuai.
  • Retensi & Penghapusan Data: Anda memiliki kendali penuh atas data Anda. Namun, PERHATIAN PENTING: Karena aplikasi ini tidak menggunakan login, jika Anda melakukan Hapus Data (Clear Data) atau menghapus (uninstall) aplikasi, maka seluruh sisa Saldo Kuota Anda akan HANGUS PERMANEN dan tidak dapat dikembalikan.
